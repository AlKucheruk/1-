# Этап 11 — обработка «Заказ клиента → Реализация товаров и услуг»

## Код

Файл модуля формы: `Код1СУТ_11_Заполнение_недостающих_Реализаций.bsl` — вставляется в модуль формы внешней обработки в конфигураторе.

## Копирование формы

1. Возьмите внешнюю обработку **этапа 10** (та же схема: файл Excel, даты, таблица, банчи, лог) как основу.
2. Сохраните как обработку этапа 11, при необходимости смените имя и синоним.
3. В **таблице `ТаблицаДанных`** добавьте и выведите в сетку колонки (типы — как в типовой практике этапа 10: строка / число / ссылка / булево):

| Колонка | Содержание |
|--------|------------|
| НомерСтроки, Дата, Контрагент, ИНН | Как в этапе 10 |
| **Договор** | Текст из Excel (может быть пусто → подбор договора **СПокупателем**) |
| **Соглашение** | Текст; пусто → подстановка `Реализация. Отсрочка 30 к.д.` |
| **Склад** | Текст; пусто → `Симфер КЭМТ` |
| **Номенклатура** | Текст наименования |
| **Количество** | Число |
| **ЕдИзм** | Текст (сверка с единицей номенклатуры) |
| **ЦенаСНДС** | Число |
| **Сумма** | Сумма с НДС (сверка с `СуммаДокумента` реализации) |
| **Выбрать** | Булево (отбор к загрузке) |
| КонтрагентСсылка, НайденКонтрагент, ДоговорСсылка, НайденДоговор | Как в этапе 10 |
| **СоглашениеСсылка**, **НайденоСоглашение** | Ссылка + булево |
| **СкладСсылка**, **НайденСклад** | Ссылка + булево |
| **НоменклатураСсылка**, **НайденаНоменклатура** | Ссылка + булево |
| **Статус**, **Комментарий** | Как в этапе 10 |

4. Команды и реквизиты формы: **АдресФайла**, **ИмяФайла**, **ДатаНачала**, **ДатаОкончания**, **РазмерБанчаЗагрузки**, **ОценкаСекундНаОдинПлатеж** (при необходимости переименуйте подпись в форме — код читает те же имена, что в этапе 10).
