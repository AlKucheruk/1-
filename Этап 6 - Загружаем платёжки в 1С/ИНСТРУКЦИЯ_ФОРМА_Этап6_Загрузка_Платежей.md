# Настройка формы внешней обработки (этап 6)

Исходный модуль формы: `Код1СУТ_06_Загрузка Платежек.bsl`. Имена ниже должны совпадать **буквально** (латиница/кириллица как указано).

## Реквизиты формы

| Имя реквизита | Тип | Назначение |
|---------------|-----|------------|
| `АдресФайла` | Строка (неогр.) | Адрес двоичных данных во временном хранилище после выбора файла |
| `ИмяФайла` | Строка | Имя выбранного файла; пишется в лог и связь БП–УТ |
| `ДатаНачала` | Дата | Нижняя граница фильтра по дате при предпросмотре (может быть пустой) |
| `ДатаОкончания` | Дата | Верхняя граница фильтра (может быть пустой) |
| `ЗагружатьПриходные` | Булево | Загружать строки-приходы (по умолчанию Истина, если реквизита нет) |
| `ЗагружатьРасходные` | Булево | Загружать строки-расходы (по умолчанию Истина) |
| `РазмерБанчаЗагрузки` | Число | После стольких **успешно обработанных в попытке** строк (с инкрементом счётчика банча) выводится сводка в сообщения и в файл лога; если реквизита нет или значение ≤ 0 — используется **10** |
| `ОценкаСекундНаОдинПлатеж` | Число | Секунд на одну строку «Готов» для грубой оценки времени; по умолчанию **5** (если не заполнено или ≤ 0) |
| `ТаблицаДанных` | ТаблицаЗначений | Основная таблица предпросмотра и загрузки (см. колонки ниже) |

## Колонки таблицы `ТаблицаДанных`

Состав должен покрывать поля, которые заполняются в `ЗагрузитьПредпросмотрСервер` из результата `ПрочитатьExcelВТаблицу`. Обязательно добавьте колонку отбора:

| Имя колонки | Тип (ориентир) |
|-------------|----------------|
| `Выбрать` | **Булево** — участвует в загрузке; если колонки нет, считается, что все строки выбраны |
| `НомерСтроки` | Число |
| `УИД` | Строка |
| `ТипДокумента` | Строка |
| `ОперацияУТ` | Строка |
| `Дата` | Дата |
| `НомерПП` | Строка |
| `Сумма` | Число |
| `Валюта` | Строка |
| `Организация`, `ИННОрг`, `КППОрг`, `НашСчет`, `НашБанк`, `НашБИК` | Строка |
| `Контрагент`, `ИНН`, `КППКонтр` | Строка |
| `ДокОснование`, `Назначение`, `СчетРасчетов`, `СчетКонтр`, `БанкКонтр`, `БИККонтр` | Строка |
| `СтатьяДДС`, `НаправлениеДенег` | Строка |
| `НДС_ИзТекста` | Строка |
| `СуммаНДС_ИзТекста` | Число |
| `НайденКонтрагент`, `НайденаОрганизация`, `НайденДоговор`, `НайденаСтатья`, `НайденСчетОрганизации`, `НайденСчетКонтрагента` | Булево |
| `КонтрагентСсылка` | СправочникСсылка.Контрагенты |
| `ОрганизацияСсылка` | СправочникСсылка.Организации |
| `ДоговорСсылка` | СправочникСсылка.ДоговорыКонтрагентов |
| `СтатьяДДССсылка` | СправочникСсылка.СтатьиДвиженияДенежныхСредств |
| `БанковскийСчетОрганизации` | СправочникСсылка.БанковскиеСчетаОрганизаций |
| `БанковскийСчетКонтрагента` | СправочникСсылка.БанковскиеСчетаКонтрагентов |
| `ЭтоДубль` | Булево |

После чтения Excel код не выставляет `Выбрать` — задайте начальное значение колонки в конструкторе (например Истина) или отдельной процедурой при открытии формы.

## Поведение «Оценка времени»

Команда `ОценитьВремяЗагрузки` вызывает серверную функцию `ОценитьВремяЗагрузкиСервер()` и показывает одно окно предупреждения с текстом: число строк «Готов», размер банча, число банчей, оценка длительности по формуле «число готовых × ОценкаСекундНаОдинПлатеж».

**Важно (ошибка «несоответствие типов, параметр 3»):** у `ПоказатьПредупреждение` третий параметр — **число** (таймаут в секундах, `0` = без авто‑закрытия), а не заголовок окна. Неверный вызов: `ПоказатьПредупреждение(, ТекстОценки, "Заголовок")`. После правок **обновите модуль формы внешней обработки из файла** `Код1СУТ_06_Загрузка Платежек.bsl`, иначе в базе останется старая строка и ошибка повторится.
