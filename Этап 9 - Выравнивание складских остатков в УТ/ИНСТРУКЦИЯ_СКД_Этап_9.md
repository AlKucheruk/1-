# Инструкция СКД (Этап 9)

## 1) Создайте новый вариант отчета в СКД

- Источник данных: запрос из файла `Запрос_УТ_Этап9_поток_движений_по_складу_для_СКД.txt`.
- Параметры запроса:
  - `ДатаНач` (тип Дата)
  - `ДатаКон` (тип Дата)
  - `Склады` (тип Список значений / массив ссылок на склады)

## 2) Настроить структуру вывода

- Только детальные записи.
- Без группировок.
- Без итогов.
- Порядок полей в таблице:
  1. `Период`
  2. `Склад`
  3. `КодНоменклатуры`
  4. `Номенклатура`
  5. `Регистратор`
  6. `Операция`
  7. `Количество`

## 3) Сортировка

Сортировать по:

1. `Склад`
2. `КодНоменклатуры`
3. `Период`
4. `Регистратор`

## 4) Режим выгрузки

- Рекомендуемый период запуска: 1 месяц.
- При небольшом объеме можно запускать квартал.
- Выгружать в `xlsx` (или `csv`) без дополнительных форматирований.

## 5) Проверка перед анализом в Python

Проверить, что:

- нет группировочных строк;
- на одну строку приходится одно движение;
- колонки называются строго как в списке выше;
- период выгружен только в заданных границах `ДатаНач` - `ДатаКон`.
